Comparative Analysis of ML Models in Predicting CAD

Collins Kwasi Apenteng & Mbye Sallah (May 2024)
Data Preprocessing and Analysis.

Introduction

Early diagnosis of coronary artery disease (CAD) enables timely medical interventions that can lessen the disease's impact, enhance patient outcomes, and save lives. This has prompted significant research in using Machine Learning (LM) algorithms to aid early diagnosis of CAD. Among the ML algorithms used, Garavand et al. (2023) document that random forest (RF), linear regression (LR), neural networks (NNs), support vector machine (SVM), and K-nearest network (KNN) are the most applied. However, the performance of these algorithms is mixed among different studies. A comprehensive literature review of studies on ML algorithms for CAD by Alizadehsani et al. (2019) reveals that the accuracy rates and sensitivities of the ML algorithms for CAD depend on the sample data, sample size, location of the data, performance metric, and ML methodology employed. For instance, while Jabbar and Deekshatulu (2013) found that the KNN classifier yields a predictive accuracy rate of 100%, Acharya et al. (2017) found 98.5% accuracy rate. A similar divergence in the accuracy rate for SVM and classifiers exists. We extend the analysis by analyzing and comparing five classifiers namely Logistic Regression, K-Nearest Neighbors (KNN), Naive Bayes, Support Vector Machine (SVM), and Random Forest using the Sani dataset uploaded on Canvas.

Data Description and Preprocessing

We use the Z-Alizadeh Sani dataset from the University of California, Irvine Machine Learning Repository. The dataset has information on 303 individuals of whom 216 are classified as CAD and 87 as normal. A CAD patient is an individual with at least 50% diameter narrowing. Individuals with a diameter narrowing of less than 50% are classified as normal. The dataset has 56 features containing demographic and health information about the 303 individuals. Not all these features are relevant for predictive algorithms. We therefore follow a feature selection criterion proposed by Kursa and Rudnicki (2011) to select the most relevant features for our analysis.

Kursa and Rudnicki (2011) showed that using a Random Forest algorithm to identify and rank the most relevant features for predicting the outcome yields reasonable accuracy. We first predict CAD using all the features in the dataset. Next, we calculate the variable importance measures for each predictor in the model using the average decrease in model accuracy on the out-of-bag samples when a given variable is excluded from the model. We then sort the features based on their importance scores and extract the top 15 features. Thus our final Sani dataset has 16 variables with 303 patients.
